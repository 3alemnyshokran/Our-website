First Letters

Let's start with some basic Arabic letters. Each letter can be written in different forms depending on its position in the word.

  • ا (Alif) - Similar to 'A'
  • ب (Baa) - Similar to 'B'
  • ت (Taa) - Similar to 'T'

Practice Question 1

What is the name of this letter: ا?

Hint: This letter makes the 'A' sound in Arabic.
